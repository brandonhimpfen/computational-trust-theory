# Research Questions

## Core Questions

1. What mathematical properties should computational trust satisfy?
2. How should trust differ from confidence?
3. How should provenance influence trust?
4. How should independent evidence combine?
5. How should contradictory evidence reduce trust?
6. How should uncertainty propagate through evidence chains?
7. How should source reliability be represented?
8. Can trust be propagated across graphs?
9. How should temporal freshness affect trust?
10. Can computational trust be calibrated and empirically validated?

## Long-Term Questions

- Can a general trust algebra be defined?
- Can computational trust become domain-independent?
- What benchmark datasets are needed?
- What properties should every implementation satisfy?
